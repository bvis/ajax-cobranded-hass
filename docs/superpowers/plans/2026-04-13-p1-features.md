# P1 Features Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement Force Arm services, icons.json, missing binary sensors, event platform, and logbook integration.

**Architecture:** Each feature is a self-contained task. Force Arm adds custom services to `__init__.py` backed by existing `SecurityApi`. Missing binary sensors extends the existing `_DEVICE_TYPE_SENSORS` mapping. Event platform adds a new HA platform that receives parsed FCM events from `notification.py`. Logbook hooks into the same events.

**Tech Stack:** Home Assistant EventEntity, custom services via `services.yaml`, FCM protobuf parsing, logbook `async_describe_events`.

```python
    @staticmethod
    def _parse_event_tag(raw: bytes) -> str | None:
        """Extract event type from raw protobuf push notification data.

        The event tag is typically in field 3 (varint) of the push notification proto.
        We scan for known varint patterns to extract the tag.
        """
        # Simple protobuf varint extraction: look for field 3 (tag byte 0x18)
        for i, byte in enumerate(raw):
            if byte == 0x18 and i + 1 < len(raw):
                tag_value = raw[i + 1]
                event_type = PUSH_EVENT_TYPES.get(tag_value)
                if event_type:
                    return event_type
        return None

    @staticmethod
    def _parse_event_data(raw: bytes) -> dict[str, Any]:
        """Extract additional event data from raw protobuf.

        Returns whatever metadata we can extract (device name, user, etc.).
        This is best-effort — unknown fields are silently ignored.
        """
        data: dict[str, Any] = {}
        # Extract embedded strings (potential device names, user names)
        # Strings in protobuf are wire type 2 (length-delimited)
        strings_found: list[str] = []
        i = 0
        while i < len(raw) - 1:
            wire_type = raw[i] & 0x07
            if wire_type == 2 and i + 1 < len(raw):
                length = raw[i + 1]
                if i + 2 + length <= len(raw):
                    try:
                        s = raw[i + 2 : i + 2 + length].decode("utf-8")
                        if s.isprintable() and len(s) > 1:
                            strings_found.append(s)
                    except UnicodeDecodeError:
                        pass
                    i += 2 + length
                    continue
            i += 1
        # Heuristic: first short string is often device name, second is user
        if len(strings_found) >= 1:
            data["device_name"] = strings_found[0]
        if len(strings_found) >= 2:
            data["user_name"] = strings_found[1]
        return data
```
